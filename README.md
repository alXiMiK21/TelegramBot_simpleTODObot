## Бот в тг для TODO

Простой бот в тг, у которого есть 3 функции:
- Вывести весь список заданий
- Добавить задачу
- Удалить задачу
- Отредактировать задачу
- Посмотреть, сколько сделал

#### Что необходимо для запуска:
1. Скачать все библиотеки из *requirements.txt*
2. Создать файл **.env** на подобии **.env.example**
3. В папке с проектом запустить `python app/main.py`
